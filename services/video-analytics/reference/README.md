# reference/ — эталон поведения видеоаналитики

Здесь лежит **рабочий PoC** `motion-log.html` — браузерный прототип, на котором
проверена логика видеоаналитики (MediaPipe `PoseLandmarker`, события и действия,
ROI-зоны и % покрытия, эвристика «белого халата»).

**Это не код для деплоя.** Файл — источник истины по поведению (пороги, размеры
окон, формулы), который переносится в Python-воркер `app/` в рамках эпика E4.
Спецификация порта — `docs/07_VIDEO_ANALYTICS.md`.

Открыть: любой браузер → СТАРТ → разрешить камеру. Лево/право — анатомические
(относительно человека), кадр зеркалится как селфи.
