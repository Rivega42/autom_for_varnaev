# ingest-sensors

Воркер: подписка на MQTT, парсинг payload ESPHome, запись показаний в БД,
контроль «тишины» узлов и генерация событий по порогам. Эпик E2.

- Python-пакет: `ingest_sensors` (уникальное имя — чтобы сервисы не
  сталкивались при общем прогоне тестов).
- Контракт MQTT — `docs/08_MQTT_CONTRACT.md`; модель `Reading` — `shared/`.
