# shared

Общие Python-модели для всех сервисов: единый конверт API и схемы
событий/заданий/артефактов — чтобы сервисы не расходились в форматах.

- Пакет: `monitoring_shared` (устанавливаемый, `pyproject.toml`).
- Каркас — E0.7; наполнение моделями — E1.
- Контракты-источники: `docs/03_API_CONTRACT.md`, `docs/04_DATA_MODEL.md`.
