# grafana

Дашборды поверх TimescaleDB: ряды датчиков и лента событий. `provisioning/` — datasource, дашборды и алерты как код. Эпик E7.

Алерты (`provisioning/alerting/rules.yaml`) визуально дублируют БД-логику датчиков: «превышение порога» и «датчик молчит». Это сигнал в интерфейсе Grafana, а не источник истины — события по-прежнему пишет `ingest-sensors` в таблицу `events`.
