Спецификация оборудования объекта

Система мониторинга помещений: видеонаблюдение с распознаванием и видеозаписью + Zigbee-датчики среды. Схема подключения и перечень оборудования (BOM). Количества — под типовой объект (до 20 камер; ~6 сухих помещений; 2 холодильные камеры + 1 морозильник), уточняются по факту.

Схема подключения

Сервер подключён напрямую к роутеру. PoE-коммутатор питает и коммутирует только камеры (и, в варианте A, Zigbee-координатор). Zigbee-датчики образуют mesh и приходят на сервер через координатор — вариант A (по сети/PoE) либо B (по USB).

Примечания для заказчика

  • Каждый датчик заводится в системе (идентификатор node_id) до подачи питания, иначе показания не принимаются.
  • ПО Zigbee-контура (Zigbee2MQTT + адаптер топиков) работает на сервере в контейнерах — отдельной закупки не требует.
  • Итоговые количества (камеры, датчики воздуха, зонды камер) уточняются по числу помещений и камер холода на объекте.
  • Видеозапись и распознавание работают на одном сервере: запись ведёт NVR, распознавание берёт только нужные фрагменты записи (без нагрузки 24/7).
